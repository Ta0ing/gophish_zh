# gophish_zh
汉化Gophish意味着将其用户界面和相关文档翻译成中文，以便中文用户能够更容易地理解和使用这个工具。汉化工作可能包括翻译安装指南、用户手册、错误消息、以及Web界面中的所有文本。
> 参考：https://github.com/gophish/gophish
